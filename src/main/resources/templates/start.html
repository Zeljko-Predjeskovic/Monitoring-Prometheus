Get Started

Use social media API

You can test the social media API on your projects by fetching from your project to our public resources

Create your own person with a POST:

https://predjeskovic.com/socialmedia/people

Requested Body: {

"firstName:" "firstName",

"lastName:" "lastName",

"username:" "username",

"email:" "email" }

Find the person your created by typing the username from your creation with a GET:

https://predjeskovic.com/socialmedia/people/username

Watch all People who made an insert by creating a get request to see all the People with a GET:

https://predjeskovic.com/socialmedia/people
